Truly challenge every assumption and idea in this artifact. Work through the decision tree branch by branch, resolving dependencies before moving forward. Offer a recommended answer with each question.

When the codebase can answer a question, explore it directly instead of asking.

Use the AskQuestion tool to batch related decisions. Update the artifact in place as branches close. Stop when every section is specific, every decision is committed or flagged as an open question, you've fully reached a shared understanding of the project with the user, and the artifact is ready to hand off.
